Adds a Terms of Service field for use in Gravity Forms

== Description ==

IMPORTANT: 
**You must have [Gravity Forms](http://gravityforms.com "Purchase Gravity Forms") installed. Click [here](http://gravityforms.com "Purchase Gravity Forms") to purchase [Gravity Forms](http://gravityforms.com "Purchase Gravity Forms")**

This plugin adds a Terms of Service field that can disable the Submit button that will re-enable when the user scrolls through the Terms of Service.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `gftos` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Begin using in your Gravity Forms
